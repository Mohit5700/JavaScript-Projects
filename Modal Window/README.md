# Modal Window

## Description
"Modal Window" is a simple web-based project that demonstrates how to create a modal window using HTML, CSS, and JavaScript. The modal can be triggered by clicking on buttons and closed by clicking the close button, clicking the overlay, or pressing the Escape key.

## Usage
- Click any of the "Show modal" buttons to open the modal window.
- To close the modal window:
  - Click the close button (&times;) inside the modal.
  - Click the overlay outside the modal.
  - Press the Escape key on your keyboard.

## Features
- Modal window triggered by button clicks.
- Multiple ways to close the modal:
  - Close button within the modal.
  - Clicking the overlay.
  - Pressing the Escape key.
- Responsive design using CSS.
- Smooth animations and transitions.

## Technologies Used
- HTML
- CSS
- JavaScript
